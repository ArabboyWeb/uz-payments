# Changelog

## 0.1.0

- Initial monorepo foundation.
- Added `@uz-payments/core`.
- Added MVP `@uz-payments/payme` provider.
- Enforced Payme 12-hour transaction timeout natively in SDK.
- Made `@uz-payments/core` audit redaction recursive for arrays and nested objects.
- Added Express and Next.js App Router adapters.
- Updated examples with strict Payme `-2` state mapping and Cancel `reason` propagation.
- Added examples, documentation, full JSON-RPC test suites, sandbox harness, and CI.
